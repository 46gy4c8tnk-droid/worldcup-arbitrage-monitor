# 世界杯跨平台套利监控

*[English](README.md) | 中文*

监控**澳洲主流博彩商的世界杯 1X2(胜平负)赔率**,检测跨平台套利(surebet),
用 **10,000 AUD 模拟账户**自动下单,开赛后按锁定赔付结算,并绘制收益率曲线。

> ⚠️ **纯模拟资金,本工具不进行任何真实下注**,仅供学习与研究。

## 功能

- **真实赔率**:通过 [The Odds API](https://the-odds-api.com)(合法聚合澳洲博彩商赔率:
  Sportsbet、TAB、Ladbrokes、Neds、PointsBet、Unibet、Betr、PlayUp 等)。
- **套利检测**:某场比赛若 `1/最优赔1 + 1/最优赔2 + 1/最优赔3 < 1` 即为套利。
- **模拟账户**:10,000 AUD,**归零自动重置为 10,000**(重置次数会记录)。
- **自动下单**:利润率达标的机会按 `1/赔率` 比例分配注金,使各结果赔付一致;
  利润在下单瞬间锁定,开赛时结算入账。
- **下单助手(半自动/手动)**:输入计划总投入 → 自动算出每家该下多少、保底盈亏,
  并给出各博彩商官网直达链接;你手动下完后一键记入模拟账户。
- **市场总览**:展示**所有**比赛离套利有多近(含负值),每次抓取刷新。
- **收益率曲线、下单历史、事件日志、桌面提醒。**
- **中英双语**:右上角按钮一键切换。

## 快速开始

```bash
pip install -r requirements.txt
python app.py
```

打开 **http://127.0.0.1:8788**。

1. 到 [the-odds-api.com](https://the-odds-api.com) 免费注册拿 API key;
2. 在**设置**面板填入并保存(存于 `config.json`,该文件已被 git 忽略,key 不会外泄);
3. 世界杯赛事 key 会自动识别。

## 抓取频率与配额(重要)

**不要直接爬取 Sportsbet/TAB 等网站**——违反服务条款且有 IP 封禁风险,本项目走合法聚合 API。

The Odds API 免费档 **500 积分/月**,`h2h` + `au` 一次轮询消耗 **1 积分**:

| 间隔 | 月消耗 | 评估 |
|---|---|---|
| 2 小时 | ~360 | ✅ 安全 |
| 90 分钟 | ~480 | ⚠️ 接近上限(默认) |
| 30 分钟 | ~1440 | ❌ 超额 |

内置保护:间隔硬下限 60 秒;429 限流指数退避;瞬时网络抖动快速重试;界面实时显示剩余配额。

## 模拟账户规则

- 单组投入 = `min(余额 × 投入比例, 单组上限)`,按 `1/赔率` 在各结果间分配;
- 同一场比赛持仓期间不重复下单;
- 在**开赛**(非比赛结束)时结算——套利赔付与赛果无关,各腿下完利润即锁定;
- 净值 = 可用余额 + 持仓锁定赔付,曲线按净值绘制。
